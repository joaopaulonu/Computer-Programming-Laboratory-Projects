## 📄 Overview

This repository contains a collection of **laboratory exercises and projects developed in C**, focusing on **procedural programming, algorithm efficiency, and low-level system interactions**.

The projects were developed during the **Programação de Computadores (Computer Programming)** course in the **Computer Engineering program at Pontifical Catholic University of Campinas (PUC-Campinas)**.

This repository serves both as **academic documentation** and as a foundational part of my **software engineering portfolio**.

---

## 🧠 Why C?

**C** is a powerful general-purpose programming language that provides low-level access to memory and a clean syntax. It is the foundation for modern computing.

### Learning C allows for:

- Understanding **memory management** (pointers, allocation, and deallocation).
- Developing **high-performance** applications.
- Gaining insight into how **Operating Systems** and compilers work.
- Building a solid foundation for learning other languages like Java, C++, and Python.

---

## 🚀 Key Skills Demonstrated

The projects in this repository demonstrate:

- **Algorithm Design:** Problem-solving using structured logic and flow control.
- **Memory Management:** Efficient use of pointers and dynamic memory allocation (`malloc`, `free`).
- **Data Structures:** Implementation of arrays, structs, linked lists, and file handling.
- **Debugging & Optimization:** Using tools like GDB and Valgrind to ensure code reliability.
- **Technical Documentation:** Writing clean, readable code with consistent naming conventions.
